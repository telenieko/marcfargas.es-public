Prohibido endeudarse en 2011

Una de las flamantes medidas que incluye el Real Decreto-ley 8/2010, de 20 de mayo, por el que se adoptan medidas extraordinarias para la reducción del déficit público. Es la prohibición de concertar operaciones de endeudamiento a largo plazo hasta el año 2012. A casi cualquiera que le pongan esto en frente lo primero que piensa es: ¿Cuanto tiempo tengo para endeudarme? y si tienes mucho tiempo la medida no sirve para nada (o puede ser contraproducente)....
